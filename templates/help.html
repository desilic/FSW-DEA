User Guide

Introduction

This a user guide & quick start page about the fsw-dea tool.

Input

The input file should in an xlsx format. The data requires rows and columns corresponding to samples and features, table headers and column names are essential. You can upload your local files or use the example data set here to have a quick start.

Output

After uploading the data files and entering the parameters, you can click the submit button to begin the training. After running, figures and tables can be seen.
The download button will also be displayed where you can get the detailed evaluation metrics table.

Example

There is a simulation data set with 500 samples in 50 dimensions. You can download the data set and have a quick start by entering the parameters of n and r with n equal to 2 and r equal to 100.
The first 48 variables are independently identically distributed in this data set and obey one normal distribution. The 48th and 49th variables obey a two-dimensional normal distribution and are divided into two sample groups.
